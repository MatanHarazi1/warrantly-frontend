# Warrantly 📄⚙️

### [🔗 קישור לאפליקציה החיה ב-Vercel](https://warrantly-frontend-bkgo.vercel.app)

---

## 📝 סקירה כללית
**Warrantly** היא אפליקציית Full-Stack מתקדמת המהווה פתרון חכם, מרכזי ומבוטח לניהול, מעקב ושימור תעודות אחריות וקבלות של מוצרים ביתיים ועסקיים.

## ⚠️ הבעיה שהפרויקט פותר
אנשים נוטים לאבד קבלות פיזיות מנייר, לשכוח מתי פג תוקף האחריות של מוצרי החשמל שלהם ומתי צריך לבצע טיפול תקופתי במכשירים יקרים. Warrantly מרכזת הכל במקום אחד מאובטח ונגיש.

## 🎯 קהל היעד
משתמשים פרטיים המחזיקים במכשירים טכנולוגיים/מוצרי חשמל ביתיים יקרים ומעוניינים בסדר וארגון דיגיטלי, וכן עסקים קטנים המעוניינים במעקב ציוד.

## ⚡ מתחרים ובידול
* **מתחרים:** ניהול ידני באקסל, פתקאות בטלפון או שמירת קבלות פיזיות במגירה.
* **בידול:** Warrantly מספקת חוויית משתמש ייעודית, קישור ישיר בין המכשיר לבין משימות או בדיקות מעבדה נדרשות, מנגנון **סורק קבלות חכם (OCR)** אוטומטי, ומערכת הרשאות (RLS) קשוחה המגנה על המידע האישי.

---

## 🛠️ טכנולוגיות וארכיטקטורה (Tech Stack)
* **Frontend:** React (Vite), React Router DOM, SweetAlert2.
* **Backend & Database:** Supabase (PostgreSQL), Supabase Auth.
* **Cloud Storage:** Supabase Storage Buckets לניהול קבצים פיזיים.
* **AI & Smart Features:** Tesseract.js (מנוע OCR מבוסס בינה מלאכותית לחילוץ נתונים).

---

## 📊 רשימת שירותים חיצוניים ואינטגרציות (דרישה 4)

| שירות | סוג | למה הוא משמש באפליקציה |
| :--- | :--- | :--- |
| **Supabase Auth** | אוטנטיקציה | רישום וניהול משתמשים מאובטח, הגנה על נתיבים (Protected Routes). |
| **Supabase Database** | מסד נתונים (PostgreSQL) | שמירת נתוני המכשירים (`items`), משימות (`tasks`) ופרטי משתמשים (`profiles`). |
| **Supabase Storage** | לוגיקת שרת וקבצים | שמירת קבצי המקור הפיזיים (צילומי הקבלות והתמונות) ב-Bucket ציבורי ומאובטח. |
| **Tesseract.js API** | שירות AI / קריאת API | מנוע ראייה ממוחשבת (OCR) שרץ בצד לקוח ומחלץ טקסט, חברות ומחירים ישירות מצילום הקבלה. |
